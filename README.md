# EcoCareer.net

Yenilenebilir enerji sektöründe kariyer fırsatları sunan modern bir web platformu.

## Özellikler

- Modern ve responsive tasarım
- Kariyer fırsatları listesi
- Eğitim ve sertifika programları
- Yeşil Kariyer Puanı sistemi
- Eko-Jeton ödül sistemi
- İnteraktif kullanıcı arayüzü
- İletişim formu

## Teknolojiler

- HTML5
- CSS3
- JavaScript
- Font Awesome ikonları

## Kurulum

1. Repoyu klonlayın
2. index.html dosyasını bir web tarayıcısında açın
